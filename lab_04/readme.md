1. Дана функция `(defun mystery (x) (list (second x) (first x)))`.
Какие результаты вычисления следующих выражений?

    * `(mystery (one two))`
    * `(mystery (last one two))`
    * `(mystery free)`
    * `(mystery one 'two))`

Как бы назывался роман Р.Брэдбери "+451 по Фаренгейту" в системе по Цельсию?
